# 화상키보드

## 실행
```
git clone "https://github.com/LeeSangMin1029/virtual_keyboard.git"
npm i && npm run dev
```

화상키보드의 추가 기능 구현 및 확장

1. 키보드의 기능을 대부분 수행하도록 제작
- 한글, 영어 키를 입력할 수 있는 상태에서는 각각에 맞는 언어를 출력

2. 펑션키(f1~f11) 눌리지 않도록 설정
- 오른쪽 마우스 금지

3. 3D처럼 보이는 css 적용
- 키 입력 및 한/영 키를 눌렀을 때 약간의 애니메이션 실행
